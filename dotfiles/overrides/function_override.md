## datauri
- Creates data uri from a file.
- Converts the file into a base 64 encoded data string and creates a uri to be easily shared.

## delete-files
- Finds and deletes all files matching given pattern along with the the annoying autogenerated `.DS_Store` files recursively from directory.

## mkcd
- Creates directories for each given arguments. And enters into the first directory.

## hist-find
- Searches given phrases in history with color.

## find-pwd
- Find a text inside current directory recursively with color and ignore case.
